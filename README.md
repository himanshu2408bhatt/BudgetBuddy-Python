# BudgetBuddy – Your Simple Finance Tracker
BudgetBuddy is a beginner‑friendly Python application that helps individuals track monthly incomes and expenses, visualize spending patterns, and maintain better control over personal finances.

## Objective
To design a simple, user‑friendly finance tracker that empowers users to make informed financial decisions by recording transactions, analyzing budget usage, and providing meaningful summaries and visualizations.

## Features
- Add income and expenses with categories
- Automatic CSV logging of transactions
- Monthly budget comparison with overspending alerts
- Pie chart visualization of expenses by category
- Daily reminder using Python threading
